Maths Library  (v.3 04-07-12)
=============

1. This version is now compatible with the String Library.

2. As far as the user is concerned, there are no changes from v.1.
  (Some code has been moved from "auxiliary.shen" to "maths-lib.shen").

3. How to load the maths library.				

	(load "m-lib.shen")
					
   Before using any of the functions, even if you are familar with their names,
   please, read the documentation in "Maths Library" 
   (available in Word, pdf, and and MHTML format).
